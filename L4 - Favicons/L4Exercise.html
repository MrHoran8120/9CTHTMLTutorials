Step 1: Choose Your Favicon Method

Pick one of these options:

  • Option A: Use a simple .ico file
  • Option B: Use a .png file
  • Option C: Create multiple sizes (recommended)

Step 2: Get Your Favicon File

Quick method:

  1. Go to favicon.io
  2. Choose "Text" and enter a letter or emoji
  3. Download the generated files
  4. Save them in the same folder as this HTML file

Alternative: Find a small 32x32 pixel PNG image and rename it to "favicon.png"

Step 3: Add the HTML Code

Uncomment and modify the line in the <head> section:

<!-- Replace this comment with one of these: --> <!-- Option A: Basic ICO file --> <link rel="icon" type="image/x-icon" href="favicon.ico"> <!-- Option B: PNG file --> <link rel="icon" type="image/png" href="favicon.png"> <!-- Option C: Multiple sizes (best practice) --> <link rel="icon" type="image/png" sizes="32x32" href="favicon-32x32.png"> <link rel="icon" type="image/png" sizes="16x16" href="favicon-16x16.png">

Step 4: Test Your Favicon

Save the file and refresh this page. You should see your icon in:

  • The browser tab next to the page title
  • Bookmarks (if you bookmark this page)
  • Browser history
