What is a Favicon?
A favicon (favorite icon) is a small 16√ó16 or 32√ó32 pixel icon that appears in browser tabs, bookmarks, and address bars. It helps users identify your website at a glance.

1. What You'll See in Your Browser Tab

Look at your browser tab right now! You should see a small icon next to this page title. That's the favicon in action.

Favicon Demo - Lesson 4

2. Basic Favicon Implementation

The simplest way to add a favicon is with a single line in your HTML head:

<link rel="icon" type="image/x-icon" href="favicon.ico">
Note: If you don't see the favicon working immediately, try refreshing the page or clearing your browser cache. Browsers sometimes cache favicons aggressively.

3. Modern Favicon Best Practices

Modern websites should include multiple favicon formats for different devices and contexts:

<!-- Basic Favicon --> <link rel="icon" type="image/x-icon" href="favicon.ico"> <!-- Modern PNG Favicons --> <link rel="icon" type="image/png" sizes="32x32" href="favicon-32x32.png"> <link rel="icon" type="image/png" sizes="16x16" href="favicon-16x16.png"> <!-- Apple Touch Icons for iOS devices --> <link rel="apple-touch-icon" sizes="180x180" href="apple-touch-icon.png"> <!-- Android Chrome Icons --> <link rel="icon" type="image/png" sizes="192x192" href="android-chrome-192x192.png"> <link rel="icon" type="image/png" sizes="512x512" href="android-chrome-512x512.png"> <!-- Web App Manifest --> <link rel="manifest" href="site.webmanifest"> <!-- Theme Color for mobile browsers --> <meta name="theme-color" content="#2196f3">

7. Testing Your Favicon

‚úÖ Favicon Checklist

  • Check browser tab - does the icon appear?
  • Test bookmarking the page - icon shows in bookmarks?
  • Try different browsers (Chrome, Firefox, Safari, Edge)
  • Test on mobile devices
  • Check high-DPI displays (Retina screens)
  • Validate HTML with favicon links
  • Test PWA installation (if applicable)

8. Common Issues and Solutions

‚ùå Problem: Favicon not showing

Solutions:

  • Hard refresh (Ctrl+F5 or Cmd+Shift+R)
  • Clear browser cache
  • Check file path is correct
  • Ensure favicon file exists and is accessible

‚ùå Problem: Favicon blurry or pixelated

Solutions:

  • Use vector graphics (SVG) when possible
  • Create multiple sizes for different contexts
  • Design at actual size (16x16, 32x32) for pixel-perfect results

9. Quick Start Guide

üöÄ 5-Minute Favicon Setup

  1. Create or find an image (square, simple design works best)
  2. Go to favicon.io and upload your image
  3. Download the generated files
  4. Copy files to your website root directory
  5. Add the HTML code to your <head> section
  6. Test in browser - refresh and check the tab!

10. Design Tips for Great Favicons

  • Keep it simple: Complex designs become unclear at small sizes
  • Use bold colors: High contrast helps visibility
  • Avoid text: Text is usually unreadable at 16x16 pixels
  • Think brand: Use your brand colors and simple logo elements
  • Test at actual size: Always preview at 16x16 and 32x32 pixels
  • Consider context: Will it work on both light and dark backgrounds?
üí° Pro Tip: Your favicon is often the first visual element users see of your website. Make it memorable and representative of your brand!
